Política de Privacidade

Como coletamos, usamos e protegemos as informações trocadas com nosso assistente pelo WhatsApp.

Versão 1.0
Vigência 29 de junho de 2026
Serviço Bot de Segunda Via · WhatsApp

1. Quem somos

A Assusa oferece atendimento automatizado via WhatsApp para que clientes possam solicitar a segunda via de boletos bancários emitidos pelo Sicoob, sem necessidade de ligação ou deslocamento.

Este documento descreve como tratamos as informações coletadas durante esse atendimento, em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

2. Dados que coletamos

Para localizar e enviar sua segunda via, coletamos apenas o mínimo necessário:

  • Número de telefone — identificado automaticamente pelo WhatsApp.
  • CPF — informado por você durante a conversa, para localizar seus boletos.
  • Histórico de interações — mensagens trocadas com o assistente, para manter o contexto da conversa.

Não coletamos senhas, dados bancários completos, documentos de identidade físicos ou qualquer informação além do necessário para a consulta de boletos.

3. Como usamos seus dados

Os dados coletados são usados exclusivamente para:

  • Consultar e retornar boletos vinculados ao seu CPF na API do Sicoob.
  • Manter o contexto da sessão ativa para que você não precise redigitar o CPF a cada mensagem (sessão de até 30 minutos).
  • Registrar interações para fins de histórico interno e melhoria do serviço.

4. Armazenamento e retenção

O estado da conversa é mantido temporariamente em memória (Redis) pelo prazo de 30 minutos após a última interação. Após esse período, a sessão é encerrada automaticamente.

O histórico de interações é armazenado em banco de dados seguro (PostgreSQL) para fins de suporte e auditoria interna. Você pode solicitar a exclusão a qualquer momento pelo canal de contato abaixo.

5. Compartilhamento de dados

Seus dados são compartilhados apenas com:

  • Sicoob — exclusivamente para consulta dos boletos vinculados ao seu CPF, via API segura com autenticação mTLS.
  • Meta (WhatsApp) — a plataforma de mensagens utilizada para o atendimento, sujeita à própria política de privacidade da Meta.

Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins comerciais.

6. Segurança

Adotamos as seguintes medidas técnicas para proteger suas informações:

  • Comunicação com a API do Sicoob via certificado mTLS (autenticação mútua).
  • Tráfego entre o WhatsApp e nossos servidores protegido por HTTPS (TLS 1.2+).
  • Validação de assinatura de cada requisição recebida do WhatsApp (HMAC-SHA256).
  • Chaves de API internas não expostas publicamente.

7. Seus direitos (LGPD)

Conforme a LGPD, você tem direito a:

  • Confirmar a existência de tratamento dos seus dados.
  • Acessar os dados que temos sobre você.
  • Solicitar a correção de dados incompletos ou incorretos.
  • Solicitar a eliminação dos dados coletados.
  • Revogar o consentimento de uso dos seus dados a qualquer momento.

Para exercer qualquer um desses direitos, entre em contato conosco pelo canal abaixo.

8. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. A data de vigência no topo da página sempre refletirá a versão mais recente. Mudanças significativas serão comunicadas pelo próprio assistente no WhatsApp.
